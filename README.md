# RNN-v3 Starter

## Usage

1. clone this repo
2. install dependencies by running `yarn`
3. start React Native webserver by `react-native start`
4. for Android run `react-native run-android`
5. for IOS run `cd ios; pod installl` then `react-native run-ios`
